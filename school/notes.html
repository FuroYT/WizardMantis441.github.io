1.1 - Computer Programs (2)

  • Computer Program - Tells a computer, in minute detail, the sequence of steps that are needed to fulfill a task.
  • Hardware - The physical computer and peripheral devices.
  • Software - The programs the computer executes.
  • Programming - The act of designing and implementing computer programs.

1.2 - The Anatomy of a Computer (3)

  • CPU - Central Processing Unit: composed of "transistors", locates and executes the program instructions, does the math, etc.
  • Hard Disk - Used for "secondary storage", can persist without electricity.
  • Networks - Allows the computer to read data and programs from central storage locations or send data to other computers.

1.3 - The Java Programming Language (6)

  • Java - A high-level programming language.
  • Compiler - Translates the high-level instructions into the more detailed instructions, "machine code", required for the CPU.
  • Applets - Java code that can be located anywhere on the internet. EXAMPLE
  • Java Library - Makes it possible to write portable programs that can bypass proprietary operating systems.
